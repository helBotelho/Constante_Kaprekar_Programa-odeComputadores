# Constante_Kaprekar_Programa-odeComputadores
Considerando a constante de Kaprekar, este é um algoritmo que a calcula com aritmética.

Descrição
    Implementa a Rotina de Kaprekar (1949): dado um número de 4 dígitos, reorganiza os dígitos em ordem crescente e decrescente, subtrai e repete até atingir 6174 (Constante de Kaprekar) em no máximo 7 iterações.
Funcionalidades
  Validação de tipo 
  Validação de faixa 
  Validação de dígitos repetidos
  Extração de dígitos por operações aritméticas 
  Ordenação manual por comparações e trocas 
  Exibição do passo a passo de cada iteração
  Contagem total de iterações até atingir 6174.
